Chat Widget Demo

Usage Example

The chat widget is initialized with the following configuration:

{
  webhook: 'https://httpbin.org/post',
  userId: 'demo_user',
  title: 'Demo Chat',
  theme: 'light',
  primaryColor: '#4F46E5'
}

Try sending a message! (This demo uses httpbin.org which will echo back your request)

Features

  • ✅ Persistent message history (localStorage)
  • ✅ Typing indicator
  • ✅ Minimize/maximize
  • ✅ Responsive design
  • ✅ Webhook integration
  • ✅ Customizable theme
